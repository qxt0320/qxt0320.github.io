未来规划

大三

大三大家开始有了比前两年更清晰的规划：直接工作？考研？考公？

对于我，可能更加倾向于保研与考研两手抓。

因此，在大三上可能更加注重课业的学习，争取让成绩更上一个台阶；大三下，在保证课业不被耽误的前提下开始进行考研的准备。

大四

如果可以顺利保研，则会提前联系导师开始自己的研0生活进行相关知识的学习，同时完成自己的毕业设计与论文

如果无法顺利保研，则全身心投入到考研中，并且在之后完成毕业设计与论文

研究生

研究生第一年在学好理论基础的同时，尝试理论与实践的深度融合，同时尝试写高水平的论文与专利。

研究生第二年积极参加科研，解决实际的生活或工程问题，希望对公司企业有积极意义。

研究生第三年参加校招或自己网上投简历，同时开始完成毕业论文。

期待与迷茫

新学期的开始，总会立下flag要好好学习，多掌握一点专业知识，积累一点实战经验，为以后的工作打好基础。
